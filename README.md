## Run the Project
All the compiled class files are contained in `App.jar` file, so you can directly run the program with this command:
```bash
java -jar out/App.jar
```

To recompile:
```bash
javac -d out src/**/*.java
jar cfm out/App.jar out/manifest.txt -C out .
```
